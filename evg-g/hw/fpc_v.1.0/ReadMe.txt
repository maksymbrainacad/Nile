  И так, пару слов о проекте!
За основу макета-дизайна был взят действующий сайт http://www.tc-pegas.ru/ созданный на основе CMS Joomla с использованием фреймворка Bootstrap 2. Он был частично изменен и доработан под техническое задание для финальных проектов на сертификацию в Brain Academy.

fpc_v.1.0 - это многостраничный сайт написанный максимально на "чистом" коде (с комментариями,  для удобства дальнейшей работы с ним) и с минимальным использованием фреймфорка Bootstrap 3 (который включает в себя файл начальных настроек внешнего вида содержимого reset.css), библиотек jquery-v1.11.3 и handlebars-v4.0.5 в соответствии с требованиями  для финальных проектов на сертификацию.

Разработка проекта велась используя систему контроля версий Github https://github.com/maksymbraincad/labs/pull/48, https://github.com/maksymbraincad/labs/tree/master/evg-g.

Создана удобная структура папок проекта (выбрана структура для дальнейшей разработки браузерного приложения на основе многостраничного сайта).

Созданы карты сайта:
 1. Для пользователей (быстрая навигация по сайту);
 2. Для поисковых роботов (sitemap.xml с перечнем ссылок на каждую страницу проекта и заданными параметрами приоритетов индексации страниц);

Сайт разработан кросс-браузерным (протестирован браузерами: Chrome 51.0.2704.84 m; Opera 38.0.2220.29; Firefox 47.0; IE 11.0.9600.17843) и одинаково отображается во всех вышеперечисленных браузерах, а так же кросс-платформенным (максимально отлажен на 4ре основные ширины экрана: 320px; 768px; 992px; 1200px и выше) с адаптацией под любую ширину экрана и отсутствием горизонтального скролла при её уменьшении (за исключением браузеров IE 11.0.9600.17843 и Opera 38.0.2220.29, появляется горизонтальный скролл, возможно из-за работы сайта на локальном сервере. Предположительно по той же причине отсутствует переход по ссылкам вложенным в кнопки в браузерах Opera 38.0.2220.29 и Firefox 47.0).

Для повышения юзабилити сайта, при больших и малых расширениях экранов некоторые блоки отображаются в более подходящих по логике пользователя для них местах. Облегчая пользователю доступ к информации и вызывая у него желание к активному действию направленному на формирование заказа на услугу.

Реализована структура разметки веб-страниц используя HTML5.

Широко используются различные типы списков. Нумерованные и ненумерованные с различными типами и способами отображения маркеров.

В разметке проекта присутствуют элементы SVG графики.

На сайт подключены и используются различные шрифты с https://fonts.googleapis.com/, а также иконочный шрифт https://maxcdn.bootstrapcdn.com/font-awesome.

Используя <iframe> к сайту подключена https://www.google.com/maps/ с нанесением на неё метки локации обьекта.

Проведена подготовительная работа по внутренней SEO оптимизации  для каждой страницы, прописаны уникальные заголовки в теге <title>, описано содержимое и ключевые слова в мета-тегах <description> и <keyword>, сайт имеет навигацию описанную в теге <nav>. В контенте сайта, с целью повышения семантического значения, некоторые ключевые слова вложены в теги <em>  и <strong>. Так же создан файл robots.txt в котором определены параметры для индексации содержимого проекта и ссылкой на sitemap.xml.

При стилизации внешнего вида блоков и контента сайта использован CSS3: многослойные градиентные заливки фонов, тени, скругление углов, 2D и 3D трансформация для различных блоков. Так же некоторые изображения оптимизированы в спрайт изображений. При отрисовке фона отдельных блоков используются паттерны изображений. Ряд элементов сайта имеют hover эффекты.

Файл стилей содержимого сайта создан при помощи препроцессора CSS - LESS. Использован укороченный синтаксис описания вложенности классов и тегов, переменные и различные математические операции с ними, различные миксины, а также цветовые функции.
При помощи менеджера задач GRUNT из файла стилей style.less скомпилирован стилевой файл styles.css и помещён в fpc_v.1.0/layout/css, но в проекте на данный момент используется файл style.less, для сохранения возможности быстрого изменения настроек внешнего вида содержимого, путем изменения значений переменных, широко описанных в файле variables.less.

Добавлена динамика сайта при помощи:
  Инструментов Bootstrap 3:
    1. Слайд Шоу, прокручивающаяся карусель изображений (на всех страницах);
    2. Форма обратной связи (на странице "контакты");
  JavaScript:
    1. Слайд Шоу, прокручивающаяся карусель изображений (на странице "мотиваторы") отрисованна на основании данных из JSON файла лежащего на сервере, полученных AJAXовым запросом. C возможностью просмотра изображений в режиме модального окна в увеличенном размере;
    2. Форма регистрации участника (на странице "регистрация участника") с полями ввода:  select / textarea / inputов типов: text, email, tel, checkbox, date, radio, button, submit. С динамически формирующейся рандомной "капчей" и возможностью её обновления.
    3. Таблица участников тренингов (на странице "отзывы") отрисовывается на основании данных полученных при AJAXовым запросе из api внешнего сервера http://api.randomuser.me/. Полученные данные обрабатываются в JSON формате и временно хранятся в localStorage, откуда вызываются при необходимости. Реализована сортировка по любому из столбцов таблицы в алфавитном и обратном порядках, а так же возможность обновить данные в таблице.
